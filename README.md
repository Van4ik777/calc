# Калькулятор шкідливих звичок

Веб-додаток на Flask для розрахунку фінансових та медичних наслідків вживання енергетиків, алкоголю та куріння.

## Функціонал

- **Розрахунок витрат** на енергетичні напої, слабоалкогольні напої та тютюнові вироби
- **Аналіз ризиків** для здоров'я залежно від інтенсивності вживання
- **Ідеї для заощаджень** - що можна купити на гроші, що витрачаються на шкідливі звички
- **Адаптивний дизайн** для мобільних та десктопних пристроїв

## Встановлення та запуск

1. Встановіть залежності:
```bash
pip install -r requirements.txt
```

2. Запустіть додаток:
```bash
python app.py
```

3. Відкрийте браузер за адресою: `http://127.0.0.1:5000`

## Структура проєкту

- `app.py` - основний файл Flask додатку
- `templates/index.html` - HTML-шаблон з інтерфейсом калькулятора
- `requirements.txt` - залежності Python

## Особливості

- Повністю адаптивний дизайн з використанням Tailwind CSS
- Інтерактивний калькулятор з можливістю вибору різних типів напоїв та куріння
- Візуалізація результатів з детальною деталізацією витрат
- Медичні попередження про можливі захворювання
- Мотиваційні ідеї для заощаджень грошей

## Технології

- Python 3.7+
- Flask
- Tailwind CSS
- Vanilla JavaScript
